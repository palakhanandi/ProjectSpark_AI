# 🚀 ProjectSpark AI

> **An AI-powered Engineering Project Mentor built using Azure OpenAI and Flask.**

ProjectSpark AI is an intelligent chatbot designed specifically for engineering students. It helps users generate project ideas, understand AI and Azure concepts, prepare reports and presentations, and receive guidance on software development—all through a modern and interactive chat interface.

---

## ✨ Features

- 💡 Engineering Project Suggestions
- 🤖 AI, Machine Learning & Deep Learning Guidance
- ☁️ Azure AI & Cloud Recommendations
- 📄 Report & Documentation Assistance
- 📊 PPT Preparation Support
- 🏆 Hackathon Project Ideas
- 📂 Project Architecture & Folder Structure
- 💬 Interactive Chat Interface

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/palakhanandi/ProjectSpark_AI.git
cd ProjectSpark_AI
```

### 2. Create a Virtual Environment

```bash
python -m venv venv
```

Activate the virtual environment.

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure Environment Variables

Create a `.env` file in the project root.

```env
AZURE_OPENAI_KEY=YOUR_AZURE_OPENAI_KEY
AZURE_OPENAI_ENDPOINT=YOUR_AZURE_OPENAI_ENDPOINT
AZURE_OPENAI_DEPLOYMENT=YOUR_DEPLOYMENT_NAME
AZURE_OPENAI_API_VERSION=2024-02-15-preview
```

### 5. Run the Application

```bash
python app.py
```

or

```bash
python api/index.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
